## 실행 방법
1. 코드를 다운로드하거나 Fork 하여 저장
2. 터미널에서 코드가 저장된 폴더로 이동하여 make 입력
3. ./MineSweeper 입력하여 실행

### 구현, 테스트 환경
- Ubuntu 18.04.5 LTS
